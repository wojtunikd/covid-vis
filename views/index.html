Relationship between the median age of a population and the number of Covid-19 deaths per million

Description

This scatter plot investigates a possible correlation between the median age of the population and the Covid-19 mortality measures as deaths per million. Shapes represent continents, while colour saturation - cases per million. Hover over the mark to read the details. Toggle the trend line and country names using button switches below.
